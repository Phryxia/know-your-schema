**Know Your Schema** is a simple GraphQL schema inspector.
It runs on client side only, therefore no server is required.

**Know Your Schema** adopts [AGPL](https://choosealicense.com/licenses/agpl-3.0/).

# Features

- Adds your graphql file to [`IndexedDB`](https://developer.mozilla.org/en-US/docs/Web/API/IndexedDB_API) on your local system
- Inspect each graphql entities based on graphql [descriptions](https://graphql.org/learn/schema/#descriptions)
